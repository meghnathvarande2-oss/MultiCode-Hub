# 🧮 Simple Calculator in C

This is a basic calculator program written in C.  
It performs simple arithmetic operations using **switch-case**.

---

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Error handling for divide by zero
- User-friendly input/output

---

## 🛠️ Technologies Used

- C Programming Language
- Standard Libraries:
  - stdio.h
  - conio.h

---

## 📋 How It Works

1. User enters the first number
2. User selects an operator (+, -, *, /)
3. User enters the second number
4. Program uses **switch-case** to perform the operation
5. Displays the result
